1.安装
  1. 安装包下载地址：https://cdn.npm.taobao.org/dist/git-for-windows/v2.29.2.windows.2/Git-2.29.2.2-64-bit.exe
  2. 安装到D盘Git文件夹或默认文件夹

4.Git提交修改
  1. 在learngit文件夹中新建readme.txt文件，内容：
    Git is a version control system.
    Git is free software.
  2. 把readme.txt添加到暂存区
    $ git add readme.txt
  3. 把readme.txt提交到版本库
    $ git commit -m "wrote a readme file" [master (root-commit) eaadf4e] wrote a readme file
    1 file changed, 2 insertions(+)
    create mode 100644 readme.txt
5.Git提交到远程库
  1. 创建SSH Key。
    在用户主目录下，看看有没有.ssh目录，如果有，再看这个目录下有没有id_rsa和id_rsa.pub这两个文件，如果已经有，可直接跳到下一步。如果没有，打开Shell（Windows下打开Git Bash），创建SSH Key：
    $ ssh-keygen -t rsa -C "123456@qq.com"
  2. 在User/jxl10下找到.ssh文件夹,打开id_rsa.pub,复制全部内容(公钥),在github新建SSH Keys, 然后在git中运行
    $ git remote add origin git@github.com:ji/learngit.git
  3. 如果报错,运行 $ git remote rm origin 删除后重新关联
  4. 把本地库推送到github仓库
    $ git push -u origin master
6.常用Git命令
  1. 查看git状态$ git status
  2. 查看git修改内容$ git diff readme.txt
  3. 查看git提交记录以及修改内容$ git log和git lg --pretty=oneline
  4. 回退到上个版本$ git reset -hard HEAD^，回退到上上个版本$ git reset -hard HEAD^^
  5. 回退到指定版本$ git reset -hard a41s5，-hard后是版本id，在$ git log中查找
  6. 如果已经关闭Git Bash，查看版本提交记录命令是$ git reflog
